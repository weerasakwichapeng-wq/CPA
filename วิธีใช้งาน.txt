วิธีเปิดใช้งาน
─────────────────────────────────────────────────────────
  ดับเบิลคลิกไฟล์:  เปิดเว็บไซต์.bat

  เซิร์ฟเวอร์จะเริ่มทำงานที่ http://localhost:8765
  เบราว์เซอร์จะเปิดเว็บไซต์อัตโนมัติ
  กด Ctrl+C ในหน้าต่าง Console เพื่อหยุดเซิร์ฟเวอร์


ฟีเจอร์
─────────────────────────────────────────────────────────
  📊 แดชบอร์ด          - สรุปข้อมูลรวม สถิติ กราฟ
  👥 เกษตรกร           - รายชื่อเกษตรกร/แปลง ค้นหา/กรอง 184 รายการ
  🗺️ แผนที่ดาวเทียม    - Polygon บนภาพถ่ายดาวเทียม (Esri)
                          • พื้นที่ให้ผลผลิต 122 แปลง (สีเขียว)
                          • เอกสารสิทธิ์ 178 แปลง (สีส้ม)
  📄 เอกสารสิทธิ์      - คลังเอกสารสิทธิ์ที่ดิน คลิกดูบนแผนที่
  🔍 ตรวจสอบย้อนกลับ   - Traceability: ค้นด้วย FMU / ชื่อ / เลขเอกสาร
  ➕ บันทึกข้อมูลใหม่  - เพิ่ม/แก้ไขเกษตรกรใหม่ แนบไฟล์ PDF/รูปเอกสาร
                          ส่งออกข้อมูลเป็น JSON ได้


โครงสร้างไฟล์
─────────────────────────────────────────────────────────
  website/
    ├── index.html           ← หน้าหลัก
    ├── style.css            ← สไตล์
    ├── app.js               ← ตรรกะแอป
    ├── เปิดเว็บไซต์.bat     ← ตัวเปิดเซิร์ฟเวอร์
    ├── วิธีใช้งาน.txt        ← ไฟล์นี้
    └── data/
        ├── members.js       ← ข้อมูล 184 สมาชิก (จาก Excel)
        ├── productive.js    ← polygon พื้นที่ผลิต (จาก KML)
        ├── productive.kml   ← ต้นฉบับ KML
        └── landtitles.kml   ← ต้นฉบับ KML เอกสารสิทธิ์
                              (โหลด+แปลงในเบราว์เซอร์อัตโนมัติ)


ข้อมูลที่บันทึกในเครื่อง (localStorage)
─────────────────────────────────────────────────────────
  • รายการที่เพิ่มในหน้า "บันทึกข้อมูลใหม่" จะเก็บใน localStorage
    ของเบราว์เซอร์ (ไม่หายแม้ปิดเบราว์เซอร์)
  • ไฟล์ที่แนบ (PDF/รูป) จะถูกฝัง base64 ใน localStorage
  • ใช้ปุ่ม "📤 ส่งออก JSON" เพื่อ backup ออกมาเป็นไฟล์


หมายเหตุ
─────────────────────────────────────────────────────────
  • แผนที่ดาวเทียมจาก Esri World Imagery (ต้องการอินเทอร์เน็ต)
  • หากเปิดด้วย file:// (ดับเบิลคลิก index.html) จะไม่สามารถ
    โหลด KML เอกสารสิทธิ์ได้ - ต้องเปิดผ่าน .bat เท่านั้น
  • รองรับ Chrome / Edge / Firefox / Safari
